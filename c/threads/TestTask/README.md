# Тестовое задание

## Task 1. Hex/Bin утилита

Написать утилиту на "чистом" (не ++) Си для конвертации файла ASCII HEX в бинарное содержимое и наоборот. Например, входной HEX файл с содержимым "30313220" преобразуется в выходной BIN файл с содержимым "012 ".

Имя входного файла и тип конвертации передается через параметр командной строки. Результат должен сохраняться в выходной файл, имя которого формируется из имени входного файла с добавлением расширения ".hex" или ".bin" в зависимости от его типа.

Например:

- вызов "conv.exe -a file-name.hex" конвертирует входной HEX файл в выходной бинарный файл с именем "file-name.hex.bin"
- вызов "conv.exe -b file-name.bin" конвертирует входной бинарный файл в выходной файл в формате HEX с именем "file-name.bin.hex"

Утилита должна выводить свое описание и инструкцию по использованию при получении параметра "-h", а также завершаться с сообщением об ошибке при получении неизвестного параметра.

Исходный код утилиты должен компилироваться с помощью gcc без ошибок и предупреждений под LINUX и Windows.

## Task 2. Поиск номера карт

Файл `trans.txt` состоит из строк. В строке в любом месте может встретиться подстрока, содержащая номер пластиковой карты в следующем формате: PAN: <номер>, где <номер> - последовательность цифр от 13 до 19 знаков без пробелов и других разделителей. При этом в файле может содержаться несколько строк, содержащих один и тот же номер карты. Требуется написать сценарий на shell, который выведет все номера карт без повторений в произвольном порядке (без префикса PAN), один номер на строку.

Пример файла `trans.txt`:

```text
Начало транзакции по карте PAN: 1236547890123
Тип транзакции – перевод
Перевод осуществляется с карты PAN: 1236547890123 на карту PAN: 9915342186763517
К карте PAN: 1236547890123 привязан счет 1236547890
На карту PAN: 9915342186763517 переведена сумма 1000.0 рублей

Начало транзакции по карте PAN: 8673542793761
Тип транзакции – перевод
Перевод осуществляется с карты PAN: 8673542793761 на карту PAN: 1236547890123
К карте PAN: 8673542793761 привязан счет 6754328765
На карту PAN: 1236547890123 переведена сумма 2000.0 рублей
```
